# Portfolio - Firmansyah Davin Falahtama

Machine Learning & AI Engineer portfolio. Dark theme modern dengan animasi, skills section, project showcase, dan contact form.

## Tech Stack
- HTML5 + CSS3 (vanilla, no framework)
- Vanilla JavaScript (typing effect, scroll animations, counters)
- Font Awesome 6 icons
- Inter Google Font

## Sections
- Hero dengan typing effect
- About dengan animated counters
- Skills dengan progress bars
- Education & Experience timeline
- Featured projects
- Contact form + social links
